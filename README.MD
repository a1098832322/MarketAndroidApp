# 项目说明

## 开发环境
* Android Studio 3.2 
* gradle 4.6

## 功能说明
> 实现功能：
> * 账号注册、登录
> * 商品查询（按商品名，按商品类别等）
> * 点击查看商品详情，查看商品缩略图详情等
> * 添加商品进购物车、查询购物车
> * 模拟购买商品、查询购买历史
